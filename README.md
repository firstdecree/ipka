# IPKA (alpha)
Master everything you want to learn, all in one place effortlessly and with complete clarity.

# ⚙️ Installations
## Github
```
git clone https://github.com/firstdecree/ipka
```

## NpmJS
```
npm install
```

## PNPM
```
pnpm install
```

# 🛠️ Setup
In order to run IPKA, it is first necessary to configure it by making adjustments to the `example.config.toml` (remove the `example.` part in the name afterwards) file. All of the necessary descriptions for each variable are already included.

# 📦 Deployment
To deploy IPKA, first create a [Vercel](https://vercel.com/) account. Then, install Vercel with the command `npm i vercel -g` and run the command `vercel --prod`. It's done! You can now use it on your websites.

# 🚀 Usage
```
node index.js
```
