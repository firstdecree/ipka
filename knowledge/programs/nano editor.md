## Nano Editor
GNU nano is a simple, easy-to-use command-line text editor for Unix-like operating systems. It is often included by default in many Linux distributions and is favored by beginners for its straightforward interface and on-screen shortcut hints.

### Key Features
- **User-Friendly:** Unlike modal editors like Vim, Nano is modeless, meaning you can just start typing right away.
- **On-Screen Help:** Common shortcuts (using the `Ctrl` key) are always visible at the bottom of the screen.
- **Syntax Highlighting:** Supports colorizing code for various programming languages.
- **Search and Replace:** Basic but functional search (`Ctrl+W`) and replace (`Ctrl+\`) tools.

### Common Shortcuts
- `Ctrl + O` - Save (Write Out) the file.
- `Ctrl + X` - Exit the editor (will prompt to save if there are unsaved changes).
- `Ctrl + W` - Search for text (Where Is).
- `Ctrl + K` - Cut the current line.
- `Ctrl + U` - Paste the cut line (Uncut).
- `Ctrl + C` - Display current cursor position and file statistics.
